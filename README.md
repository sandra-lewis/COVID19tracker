## About

This project tracks the global and country-wise spread of the Coronavirus disease around the world.

## Technologies

Project is created with:

- React v16.13.1
- React Chart.js v2.9.3
- Axios v0.19.2
- Bootstrap v4.5

## Setup

To run this project, install it locally using npm:

```
$ cd covid19-tracker
$ npm install
$ npm start
```
